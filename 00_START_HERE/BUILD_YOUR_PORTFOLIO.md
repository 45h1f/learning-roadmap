## üìç Step 2: Select Your 3 Projects

### **Project Selection Criteria:**

‚úÖ **Must Have:**
- Real-world use case (solves actual problem)
- Modern tech stack (2025 relevant)
- Deployable/Live demo
- Impressive to recruiters
- Shows variety of skills

‚ùå **Avoid:**
- Tutorial clones (another todo app)
- Outdated tech stack
- Half-finished projects
- No live demo
- Too simple

### **The Perfect Project Trio Formula:**

**Project 1: Full-Stack Application** (Shows end-to-end skills)
- Examples: SaaS platform, E-commerce, Social media app
- Must have: Frontend, Backend, Database, Auth, Deployment
- Time: 2-3 weeks

**Project 2: Specialized/Technical Project** (Shows deep expertise)
- Backend: Microservices, APIs, Data pipeline
- Frontend: Component library, Dashboard
- DevOps: CI/CD, Infrastructure automation
- AI/ML: ML model deployment, AI integration
- Time: 1-2 weeks

**Project 3: Quick Win/Creative Project** (Shows creativity & initiative)
- Examples: Automation tool, Chrome extension, CLI tool
- Must have: Unique idea, polished execution
- Time: 3-5 days

## üìç Step 5: Write Your Resume

### **Resume Writing Process:**

**Week 1: Draft**
1. Fill in basic information (contact, education)
2. List all work experience
3. Add your 3 projects from Step 3
4. List technical skills

**Week 2: Refine**
1. Rewrite bullets using the formula:
   ```
   [Action Verb] + [What] + [Tech] + [Metric]
   ```
2. Add quantifiable results everywhere
3. Tailor to target job descriptions
4. Get feedback from peers/mentors

**Week 3: Polish**
1. Proofread 3+ times
2. Check ATS compatibility (Jobscan)
3. Test different versions
4. Create PDF and .docx versions

### **Resume Sections Priority:**

**For Junior Developers:**
1. Projects (40% of space)
2. Skills (20%)
3. Experience (20%)
4. Education (15%)
5. Certifications (5%)

**For Mid-Level:**
1. Experience (50%)
2. Projects (20%)
3. Skills (15%)
4. Education (10%)
5. Certifications (5%)

### **Common Resume Mistakes to Avoid:**

‚ùå **Weak:** "Built a web application using React"
‚úÖ **Strong:** "Developed React application serving 10K+ users with 95+ Lighthouse score"

‚ùå **Weak:** "Worked on backend APIs"
‚úÖ **Strong:** "Architected RESTful API handling 50K+ requests/day with 99.9% uptime"

‚ùå **Weak:** "Used Docker for deployment"
‚úÖ **Strong:** "Containerized microservices with Docker, reducing deployment time by 75%"

### **Resume Export:**

1. **PDF Format** (Primary)
   - Use LaTeX or professional tool
   - Ensure fonts embed properly
   - Keep under 1MB file size
   - Name: `FirstName_LastName_Resume.pdf`

2. **.DOCX Format** (Secondary)
   - Some ATS systems prefer this
   - Keep formatting simple
   - Test how it looks when opened

3. **Plain Text** (Backup)
   - For copy-pasting into forms
   - Remove all formatting

## üìç Step 7: Apply & Get Hired

### **Job Application Strategy:**

**Quality > Quantity**
- 5-10 tailored applications > 100 generic ones
- Each application should take 30-60 minutes

**Application Checklist:**
- [ ] Resume tailored to job description
- [ ] Cover letter personalized (if required)
- [ ] LinkedIn profile matches resume
- [ ] Portfolio showcases relevant projects
- [ ] GitHub profile is clean and active
- [ ] Applied through multiple channels:
  - [ ] Company website
  - [ ] LinkedIn
  - [ ] Referral (if possible)
  - [ ] Recruiter contact

### **Where to Apply:**

**Job Boards:**
- LinkedIn Jobs (best for networking)
- Indeed (high volume)
- Glassdoor (company research)
- AngelList/Wellfound (startups)
- Hired, Triplebyte (vetted candidates)
- RemoteOK, WeWorkRemotely (remote jobs)
- Hackernews "Who's Hiring" (first of each month)

**Direct:**
- Company career pages
- Reach out to hiring managers on LinkedIn
- Attend meetups/conferences
- Referrals from network

### **Follow-Up Strategy:**

**After Applying:**
- Wait 1 week ‚Üí Send follow-up email
- Wait 1 more week ‚Üí LinkedIn message to recruiter

**After Interview:**
- Send thank-you email within 24 hours
- Reference specific conversation points
- Reiterate interest and fit

**Email Template:**
```
Subject: Thank you - [Position] Interview

Hi [Interviewer Name],

Thank you for taking the time to speak with me about the [Position]
```

### **Negotiation Tips:**

**When You Get an Offer:**
1. **Don't accept immediately** - Ask for 24-48 hours
2. **Research market rates** - Use levels.fyi, Glassdoor
3. **Consider total compensation:**
   - Base salary
   - Bonus/equity
   - Benefits (health, 401k match)
   - PTO
   - Remote flexibility
   - Growth opportunities
4. **Negotiate professionally:**
   - Express enthusiasm first
   - Provide market data
   - Ask: "Is there any flexibility on the base salary?"
   - Never give your current salary
5. **Get everything in writing**
