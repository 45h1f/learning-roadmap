### "I'm Stuck on a Bug for 4 Hours"

**DON'T:**
- Watch 2-hour tutorial
- Start over from scratch
- Switch to different project
- Give up

**DO:**
1. Take 15-minute walk
2. Explain problem to rubber duck
3. Google exact error message
4. Check GitHub issues
5. Ask ChatGPT with full context
6. Post in Discord with:
   - What you're building
   - What you expect
   - What's happening
   - What you've tried
   - Code snippet
   - Error message

**Rule:** Never stuck > 30 minutes without asking for help
